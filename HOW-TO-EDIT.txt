SEVENSEERS CONSULTING — HOW TO EDIT THIS SITE
=============================================

Files:
  index.html    — the whole website
  privacy.html  — the privacy page
  assets/       — logo + hero image. KEEP THIS FOLDER next to the
                  two HTML files or the logo will not appear.

No database, no login, no build step. Open a file in any plain-text
editor (Notepad, TextEdit in plain-text mode), change the words,
save, re-upload. Search for "EDIT:" to find every editable spot.

1. CURRENT MANDATES (list of live roles)
----------------------------------------
Search index.html for: CURRENT MANDATES
Each role runs from <li> to </li>. Change three things:
   <h3>Role title</h3>
   <p>One line about the role.</p>
   <a href="JOB-PORTAL-LINK-HERE">View the full posting</a>
 - ADD:    copy a whole <li>...</li> block and edit it.
 - REMOVE: delete its whole <li>...</li> block.
 - None:   delete the whole <section>, as the comment explains.


2. EMAIL, PHONE, ADDRESS

3. TESTIMONIALS — READ THIS
---------------------------
The "References" section deliberately contains NO quotes. Nothing
on this site is invented. If a client gives WRITTEN permission for
a real quote, replace the contents of one of the three boxes with
their words and name. Never add a quote nobody said.


4. EVERYTHING ELSE
------------------
All wording is ordinary text in index.html. Change any sentence —
just keep the tags around it (<p> and </p>) intact.


5. DO NOT TOUCH
---------------
The block between <style> and </style> controls colour, type and
spacing. The site's red is sampled from the logo itself.
